### Installation
```bash
uvx credential-manager-mcp
```

### Configuration
Add to your MCP client config:
```json
{
  "mcpServers": {
    "credential-manager": {
      "command": "uvx",
      "args": ["credential-manager-mcp"],
      "env": {
        "CREDENTIAL_MANAGER_READ_ONLY": "false"
      }
    }
  }
}
```
